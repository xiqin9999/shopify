Sample of Recipes from Foodly Theme

Articles in the blog help to increase your audience and generate more traffic and sales for your store.That's why you need to add more articles. The fast and easily way to increase blog posts are adding recipes. For a start you can use recipes from Foodly theme which are represent in this paper with images, texts, and tags.

Also, you can add your own recipes or use one of the following resources:

  • epicurious.com
  • simplyrecipes.com
  • iamafoodblog.com
  • foodnetwork.com
  • allrecipes.com

Now, consider in more detail how to add recipe image, tags and code from this document into your store:

  1. The first, open page for adding new blog post: 'Online Store' (Fig. 1.1 - [ 1 ]) > 'Blog Posts' (Fig. 1.1 - [ 2 ]) > 'Add blog post' (Fig. 1.1 - [ 3 ])
  2. Type the name of a recipe. You can use the name from this paper.
  3. Add text: copy the recipe code by clicking on 'COPY CODE TO CLIPBOARD' button. And then paste this code into the 'Content' field of your new article. (Fig. 1.2 - [ 4 ]).
  4. Add image: Download the image from this paper by clicking on recipe image. Load this image to the article (Fig. 1.2 - [ 6 ]).
  5. Add tags: copy tags from this paper and paste into the tags field of your article (Fig. 1.2 - [ 7 ]).
  6. Save your article (Fig. 1.2 - [ 8 ])
